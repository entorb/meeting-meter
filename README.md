# Meeting Meter App (MMA): The "Taxi Meter" for Meetings

Calculate the cost of meetings based on the number of attendees and their hourly rates.

Deployed to <https://entorb.net/meeting-meter/>

## Update dependencies

```sh
# single package
pnpm up vite
# all
pnpm up
```
